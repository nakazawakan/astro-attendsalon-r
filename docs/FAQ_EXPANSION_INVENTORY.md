# FAQ 拡充 — 資料・サイト突合インベントリ

クライアント資料（デスクトップ `00_研修データ/Sheet1.html` 等）とサイト正本 `src/data/faqs.ts`、主要ページを突き合わせたメモ。**掲載文の最終確定はクライアント合意後**に `faqs.ts` を更新する前提とする。

## 料金表記の整理（要確認）

| ソース | 内容 |
|--------|------|
| 既存 FAQ「研修費用を知りたい」 | 1 研修 120～180 分・参考価格 20 万円～、組み合わせ・人数で相談 |
| Sheet1.html（目安） | 90 分帯 15～20 万 / 120 分帯 20～30 万 などのレンジ表記 |
| 方針 | FAQ 本文では **両レンジと従来の 20 万～表記を一文で接続**し、**確定金額の断定は避ける**。最終数値・文言は Sheet と LP 他セクションと **一本化**すること。 |

## 講師資格（サイト実態との整合）

| ソース | 内容 |
|--------|------|
| 旧 FAQ | 「臨床心理士などの専門資格」 |
| `InstructorSection` / 講師紹介 | 公認心理師・産業カウンセラー等（臨床心理士の単独表記なし） |
| 方針 | FAQ は **講師紹介セクションと矛盾しない表現**に変更（公認心理師・産業カウンセラー等＋実務経験）。 |

## 無料ヒアリングと「見積無料」の関係

| ソース | 内容 |
|--------|------|
| 資料 | 30 分無料ヒアリング（介護向け記述など） |
| 既存 FAQ | 初回ヒアリング・概算見積は無料 |
| 方針 | 「見積もりは無料」の回答に **短時間の無料ヒアリングがある場合がある旨**を追記し、重複質問は増やさない。 |

## 重複・役割分担（サブページ・コラム）

| テーマ | FAQ での扱い | 誘導先（既存ルートのみ） |
|--------|----------------|---------------------------|
| 感情労働の概要 | 短い回答＋ CTA 維持 | `/column/about-emotional-labor` |
| ハラスメント研修の中身 | `/faq` で詳しめ、トップは省略可 | `/column/about-harassment-training` |
| アンガーマネジメントの中身 | 同上 | `/column/about-anger-management-training` |

| パッケージ・年間 | `/faq` のみ | `/contact/` |

## 実装メモ（コード側）

- `FaqItem` に `category`（`/faq` の見出し用）と `showOnHome`（`false` でトップ非表示）を追加。
- トップは `faqsForHome`（`showOnHome !== false`）のみ。
- `/faq` はカテゴリ順で全件表示し、任意で `FAQPage` JSON-LD を付与。
