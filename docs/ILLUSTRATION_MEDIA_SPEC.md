- **ヒーロー右イラストは `img_kaiseianeihou_01.jpg` に極力寄せる**: **濃紺のペン線**で手＋ハートの形をはっきり描き、**線の背後**に **黄橙・ピンク・ラベンダーの半透明の水彩円**を置く（霧だけで線が消える表現は不可）。構図は両手がハートを支える。

## 必須ルール（イラスト・アイコン・装飾グラフィック）

対象: ヒーロー装飾、感情労働blobs、サービス／About 系の**描画アイコン**、見出し装飾など（下記「対象外」と区分）。

1. **線画**
   - 色は純黒にしない。**くすんだ濃紺〜インィゴ**（例: `#3d4f66`〜`#4a5d78` 付近。サイトの藍系トークンと調和すること）。
   - 均一なベクター線ではなく、**手描きの揺れ・強弱**のあるライン（サインペン／細筆のイメージ）。
   - 緻密すぎず、要素は**シンボリック**に簡略化。

2. **着色**
   - 線の内側をべた塗りで埋めるのではなく、**線の背後**に**水彩風の色だまり**（にじみ・**正円にしない**）を置く。
   - 透明感があり、重なりでやや深く見える程度。**彩度は抑えたパステル〜グレイッシュ**を基調としつつ、**霞みすぎ・コントラスト不足**にならないよう、特にヒーロー等の大きい枠では**中〜やや高めの彩度**まで許容する（視認性と温かさのバランス）。
   - 線と色の位置を**わずかにずらし**、ルーズで柔らかい印象にする。

3. **背景・効果**
   - **背景は白または完全透過**（Web では原則 **透過 PNG** または透過 SVG）。
   - **box-shadow / drop-shadow を付けない**（フラット配置。写真用の微シャドウは別仕様）。

4. **余白・トーン**
   - 被写体周りに**十分な余白**。全体的に**柔らかい・温かい・公的資料の挿絵**のようなトーン。

5. **ファイル形式**
   - ラスタ: **透過 PNG**（必要なら Web 用に長辺 900〜1400px 程度へリサイズ可）。
   - ベクター: **SVG**（背景透明、不要なフィルタやドロップシャドウを入れない）。

## 対象外（本仕様とは別扱い）

次は**実写**のため、水彩手描きトーンにはしない。

- 講師ポートレート（`instructor-*.png`）
- お客様の声の写真（`voice-*.jpg`）
- About の業種写真グリッド（`about-grid-*.jpg`）
- ロゴ（`logo.png`）

これらは**撮影・既存ブランド**を優先。ただし**過度な彩度・コントラスト**は避け、サイト全体の落ち着いた印象と矛盾しないこと。

## 実装メモ（Astro / Tailwind）

- 正本のデザイン変更がある場合は、AGENTS.md の通り **DESIGN.md（正本）を先に更新**してから本ファイルと `public/images/` を更新する。
